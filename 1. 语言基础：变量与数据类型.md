
# 变量与赋值

- 变量（variables）：计算机语言中能够存储计算结果或者能表示值的抽象概念
- 直观理解：储存值
- 语法：<类型> 变量名 = 值
- 例子：

```java
int count = 3;
char ch = 'x';
boolean isUpdated = true;
int[] nums = new int[3];
```

# 变量命名

- 变量名采取LowerCamelCase（小驼峰），除第一个单词首字母小写外，其他单词首字母大写。
- 一定要取有意义的名字
- 能用中文变量名但是不建议用
- 不能是Java关键字
- 变量一定要声明了才能用
- 变量名不可重复
